# az-storage-decompress

### 1. yarn install
### 2. create .env file in root <br />
  STORAGE_NAME = <upload storage> <br />
  STORAGE_CONTAINER = <upload container> <br />
  STORAGE_KEY = <upload key> <br />
  TARGET_URL = <compressed file> <br />
 ### 3. create folder name target
 ### 4. node start.js
